# ADR-0002: Frontend em Angular 19

- Status: Aceito
- Data: 2026-07-05

## Contexto

É necessário um frontend estruturado para fluxo autenticado, tela Kanban, formulários e organização modular.

## Decisão

Adotar Angular 19 como framework principal do frontend.

## Consequências

### Positivas
- Estrutura forte para aplicações corporativas.
- Boas práticas de organização por módulos/componentes.
- Ecossistema maduro para formulários e roteamento.

### Negativas
- Maior verbosidade comparado a alternativas mais minimalistas.
- Necessidade de disciplina arquitetural para manter simplicidade.
